Requirements
------------

* [colorama](https://pypi.python.org/pypi/colorama)

### Windows-only

* [PyWin32](http://sourceforge.net/projects/pywin32)
* [WMI](https://pypi.python.org/pypi/WMI)
* [PIL](https://pypi.python.org/pypi/PIL) (for screenshotting)
